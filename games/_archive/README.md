# Archived Legacy Game Code

This directory contains original/legacy game implementations that have been superseded by the main `games/*.py` files.

These files are kept for reference but are NOT used by the game launcher.

## Active vs Legacy

| Game | Active File | Legacy Files (archived) |
|------|-------------|-------------------------|
| Snake | `games/snake.py` | `snake_game_legacy.py` |
| Pac-Man | `games/pacman.py` | `pacman_legacy.py` |
| Frogger | `games/frogger.py` | `frogger_legacy.py`, `frogger_actors_legacy.py` |
| Centipede | `games/centipede.py` | `centipede_GameMain_legacy.py`, `centipede_GameSprites_legacy.py` |
| Dig Dug | `games/digdug.py` | `digdug_Game_legacy.py`, `digdug_Classes_legacy.py` |
| Boulder Dash | `games/boulderdash.py` | `boulderdash_legacy.py` |

## Asset Directories

The following directories remain in `games/` because they contain assets (images, sounds, fonts, levels) that may be used by the active game implementations:

- `games/boulder-dash/` - Level data and images
- `games/centipede/` - Sprites and sounds
- `games/dig-dug/` - Tunnel and enemy sprites
- `games/pacman/` - Images and sounds

## Archived: 2026-01-07
